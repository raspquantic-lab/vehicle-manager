차량 운행 보고서 웹앱 v1.0 (PC 프로그램 v3.93 기준)

구성 파일
- index.html
- wasm_exec.js
- vehicle_manager.wasm
- source.go

사용 방법
1. 네 파일을 같은 폴더에 둡니다.
2. 정적 웹 호스팅(GitHub Pages, Cloudflare Pages, Netlify 등)에 폴더 내용을 올립니다.
3. 생성된 주소를 휴대폰 브라우저로 엽니다.
4. 카카오톡 TXT 파일을 선택하고 엑셀 파일 생성 버튼을 누릅니다.

개인정보
- 모든 분석과 엑셀 생성은 사용자의 브라우저 안에서 수행됩니다.
- 카카오톡 파일은 별도의 서버로 전송되지 않습니다.

주의
- index.html을 파일 앱에서 직접 열면 브라우저 보안 정책 때문에 WASM 파일을 불러오지 못할 수 있습니다.
- 정적 웹 호스팅 주소에서 실행하는 것이 가장 안정적입니다.
